## ğŸ†˜ Troubleshooting

### **No Events Showing**
**Causes:**
- All importance levels disabled
- No currencies selected
- No events scheduled today

**Solutions:**
1. Click "Reset to Default"
2. Enable at least one importance level
3. Select at least one currency
4. Check if it's a holiday

---

### **Events in Wrong Time Zone**
**Solution:**
- Events automatically display in your system's time zone
- Check your computer's time zone settings
- Refresh page after changing time zone

---

### **Can't See Actual Values**
**Reason:**
- Values appear only after event occurs
- Check event time hasn't passed yet
- Some events have delayed releases

---

### **Too Many Events**
**Solution:**
1. Disable "Low Importance" events
2. Filter to specific currencies only
3. Focus on high-impact events only

---

### **Settings Not Saving**
**Note:**
- Settings are session-based
- Reset when page refreshes
- This is intentional for flexibility
